For indicating valid entry, use the class combination is-valid,valid-feedback. For indicating invalid entry, use the class combination is-invalid,invalid-feedback. 

            
                <div>
                    <label class="form-label mt-4">Valid input</label>
                    <input type="text" value="correct value" class="form-control is-valid">
                    <div class="valid-feedback">Success! You've done it.</div>
                </div>
                <div>
                    <label class="form-label mt-4">Invalid input</label>
                    <input type="text" value="wrong value" class="form-control is-invalid">
                    <div class="invalid-feedback">Sorry, that username's taken. Try another?</div>
                </div>
