The basic HTML header components can be used as is. Further, text-muted and lead can be used to style specific typography types. 

            
                <h1>Heading 1</h1>
                <h2>Heading 2</h2>
                <h3>Heading 3</h3>
                <h4>Heading 4</h4>
                <h5>Heading 5</h5>
                <h6>Heading 6</h6>
                <h3>
                    Heading
                    <small class="text-muted">with muted text</small>
                </h3>

Different parts of speech can be developed as in the demonstration below. 

            
                <h2>Example body text</h2>
        <p>Vestibulum nec feugiat sapien. Aenean.
            <a href="#">Aliquam feugiat et.</a>
           ---
        </p>
        <p><small>This line of text is meant to be treated as fine print.</small></p>
        <p>The following is <strong>rendered as bold text</strong>.</p>
        <p>The following is <em>rendered as italicized text</em>.</p>
        <p>An abbreviation of the word attribute is <abbr>attr</abbr>.</p>

                    </blockquote>
                    <figcaption class="blockquote-footer">
                        Someone famous in <cite>Source Title</cite>
                    </figcaption>
