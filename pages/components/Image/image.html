Responsive Image Flexible


If you want the image to scale both up and down on responsiveness, set the class to responsive-flexible. 

            
                <img src="image-sample1.jpg" alt="sampleImage1" class="responsive-flexible">

Responsive Image flexible to both scale down only


If you want an image to scale down if it has to, but never scale up to be larger than its original size, set the class to responsive-scale-down. 

            
                <img src="../../../images/image-sample2.jpg" alt="sampleImage2" class="responsive-scale-down">

Responsive Image with a fixed max width


If you want to restrict a responsive image to a maximum size, set the class to responsive-fixed-max-width. 

            
                <img src="../../../images/image-sample3.jpg" alt="sampleImage3" class="responsive-fixed-max-width">

Circle Fitted Responsive Image


If you want to restrict a responsive image to a circle. set the parent div's class to responsive-circle and image to round. 

            
                <div class="responsive-circle">
                    <img src="../../../images/image-sample4.jpg" alt="sampleImage4" class="round">
                </div>
